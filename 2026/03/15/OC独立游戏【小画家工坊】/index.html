Demo参考：小画家的调色盘

1.游戏概述

  • 游戏名称：小画家工坊（暂定）
  • 游戏类型：治愈系生活模拟 + 角色扮演 + 视觉小说元素
  • 目标平台：PC（Steam），后期可考虑移动端
  • 一句话概括：玩家在“小画家工坊”中遇见七位性格迥异的小画家，通过画画互动、培养好感，最终在“精神与物质”的天平上找到属于自己的平衡。
  • 核心卖点：
    • 角色驱动的治愈体验：七位小画家对应七种存在状态，每个角色都有独特的偶遇方式、互动机制和好感度事件。
    • 二周目深度叙事：一周目是“连接”，二周目是“理解”。玩家可选择进入个人线，探索每个角色的阴影面与真实困境。
    • 玩法即理念：好感度系统、偶遇机制、线上线下双轨互动，都服务于“如何真正认识一个人”的核心命题。

2.游戏机制

  • 核心循环：探索地图 → 偶遇小画家 → 互动（一起画画/线上交流） → 提升好感度 → 解锁新剧情/新画作 → 收藏至工坊
  • 好感度系统：
    • 0-30：初识阶段，普通对话
    • 31-60：信任阶段，解锁个人故事
    • 61-90：亲密阶段，解锁“一起画画”事件
    • 91-100：羁绊阶段，触发“珍贵的画”结局事件
  • 时间与地点系统：
    • 七个时间段：清晨、上午、中午、下午、傍晚、夜晚、凌晨
    • 地图场景：小画家工坊、街头、公园、天台、图书馆、教室、河边等
    • 角色出没规则（示例）：
      • 月见：极低概率直接偶遇，主要通过线上平台互动
      • 陶陶：白天任意时段，随机街头概率出现
      • 墨言：清晨图书馆固定刷新
      • 阳阳：公园/天台，天气晴朗时高概率出现
  • 画作系统：玩家可在自己房间创作画作，可发布至“社交平台”（游戏内模拟），小画家会点赞评论，影响好感度。

  • 可自定义姓名与外型（简单2D捏脸）
  • 初始进入工坊，由Mor引导

4.剧情结构

一周目：相遇与连接

  • 玩家进入工坊 → 遇见Mor → 认识七位小画家 → 完成初始问卷 → 同类型小画家优先加好友 → 调色盘收藏至工坊 → 自由探索，逐步培养好感度
  • 结局条件：所有角色好感度达到100，触发大合影结局

二周目：理解与选择

  • 玩家可选择进入特定角色的个人线
  • 此时其他角色未加入工坊，玩家需独自寻找并建立连接
  • 每个角色线有多分支结局（救赎线/沉沦线/开放式结局）
  • 例：月见线中，玩家需自行在社交平台找到她的账号，好感度提升难度增加；达到100后可能触发“共沉沦”结局，也可能引导她遇见陶陶

5.技术需求(初步)

  • 2D立绘与场景插画（约7主角色 + NPC + 多场景）
  • 对话系统与好感度数值系统
  • 时间系统与角色调度系统
  • 简单的画板功能（玩家创作）
  • 模拟社交平台界面
